docker studys

基于阿里云服务器CentOS 操作系统,linux操作系统

首先Xshell连接一下服务器，这里我就不演示了

安装docker指令

1.yum 包更新到最新

yum update

2.安装需要的软件包

yum install -y yum-utils device-mapper-persistent-data lvm2

3.设置yum源

yum-config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo

4.安装docker,出现输入的界面都按y

yum install -y docker-ce

5.查看docker的版本，并且检查更新

docker -v

docker服务相关命令

1.启动docker服务

systemctl start docker

2.停止

systemctl stop docker

3.重启

systemctl restart docker

4.查看状态

systemctl status docker

5.开机自启

systemctl enable docker

docker镜像相关命令

1.查看镜像

docker images

2.搜索

docker search …(…替换到你需要下载的镜像）

3.拉取

docker pull …(…替换你的镜像)

4.删除

docker rmi …(…替换镜像ID)

docker容器相关命令

1.查看容器

docker ps -a

2.创建

docker run -it –name=（名字）根据镜像名称：版本

docker run -it –name=c1 centos:7 /bin/bash (举个例子)

3.进入

docker exec -it c1 /bin/bash

4.启动

docker start c1

5.停止

docker stop c1

6.删除

docker rm c1

docker rm ‘docker ps -aq’(删除所有容器，启动的容器不能删除)

7.查看容器信息

docker inspect c1
